# 6. EMA Setup

The R5 charts should use two exponential moving averages:

- **EMA 8**: short-term trend indicator
- **EMA 21**: medium short-term trend indicator

Technical interpretation:

- If price is above both EMA 8 and EMA 21, momentum is usually bullish.
- If price falls below EMA 8, short-term momentum weakens.
- If price falls below EMA 21, the trend becomes more clearly bearish.
- If EMA 8 crosses below EMA 21, bearish momentum is confirmed.

For this week, the available market evidence shows sharp declines in SPX, NDX, and IWM. Therefore, the EMA zone should be treated as an important confirmation area.
